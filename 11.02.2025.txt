Переменные 
Объявление переменных
После того как переменной будет присвоино значение,
то мы сможем использовать её в любом контесте, где может использоваться исходное значение
Объявление переменной
var qwer int
var leght, width float64
var customName string
Присваивание значений переменным
qwer = 4
leght, width = 1.2, 2.4
customName = "Name"
Использование Переменных
fmt.Println(customName)
fmt.Println("has ordered", qwer, "sheets")
fmt.Println("each with an area of")
fmt.Println(length*width, "square meters")
в одной команде можно присвоить значение сразу нескольким переменным.
Для этого перечислите имена переменных слева от = и такое же количество в правой части, разделяя их запятыми.
 qwer, qwerty = 1.2, 2,3
если значение переменной известно заранее, можно объявить переменную и присвоить ей значение в одной строке:
var qwer int = 4
Существующим переменнам можно присваивать новые значения, но эи згначения должны относится к тоу же типу.
Статическая типизация GO гарантирует, что переменной не будет случайно присвоено значение неподходящего типа.
Если значение переменной присваивается одновременно с её объявлением, тогда тип переменной в объявлении обычно не указывают.
Тип значения, присвоенного еременной, будет считаться типом этой переменной.

Нулевые значения
Если переменные объявляются без присваивания значенмия, то эти переменные будут содержать нулевое значениедля этого типа.
Для чисовых типов нулевое значение равно 0.
Для строковых переменных нелувым значением является пустая строка.
а для переменных bool - значение false
