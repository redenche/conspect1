программма - набор инструкций для компьютера, которая помгает решить определенную задачу.
pakage(пакет с пакетами) - представляет собой набор из блоков кода, выполняющих похожие операции - например, 
форматирование строк или построенние графических изображений.
import(пакет из пакета с пакетами) - в программе указываем только те покеты которые нам нужны,
- тоесть импортирует эти пакеты.
функция представляет сбой набор строк кода, которые могут выполнятся из других мест прогрмммы.
при запуске программа GO ИЩЕТ ФУНКЦИЮ С ИМЕНЕМ main и выполняет её в первую очередь т.к. она стартовая;
поэтому поэтому мы присвоили своей функции имя main.
программы go должны соблюдать определенные правила, что бы не сбивать с толку компилятор. Если вы нарушили одно из таких то компилятор сообщит об ошибке.
